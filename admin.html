Модерация отзывов

Только для владельца · клиенты не видят эту страницу

Нужна одноразовая настройка

  1. Зайди на supabase.com → создай бесплатный проект
  2. SQL Editor → вставь код из файла supabase-setup.sql → Run
  3. Settings → API → скопируй URL и anon public key в js/config.js
  4. Authentication → Users → Add user (твой email и пароль для входа сюда)
  5. Обнови сайт на GitHub

Вход в модерацию

PIN-код — без email и WhatsApp. Только ты знаешь код.

Первый раз? Запусти supabase-admin-pin.sql в Supabase. PIN по умолчанию: 472891
